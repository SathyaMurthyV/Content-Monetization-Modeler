# Content-Monetization-Modeler
The Content Monetization Modeler is a data-driven application that helps creators and media companies forecast potential YouTube ad revenue based on video performance metrics such as views, likes, comments, engagement rate , t combines exploratory data analysis (EDA), feature engineering, and regression modeling to uncover what drives content profitability — then delivers interactive insights through a Streamlit dashboard.
Features :

EDA & Visualization: Analyze trends, correlations, and patterns in YouTube video performance.
Data Preprocessing: Handle missing values, encode categorical variables, and engineer new engagement features.
Machine Learning Models: Train and compare multiple regression algorithms (Linear Regression, Random Forest, etc.) to predict ad revenue.
Model Evaluation: Assess models using R² Score, MAE, and RMSE metrics.
Feature Insights: Identify the most influential features driving ad revenue.
Streamlit App: User-friendly dashboard for live ad revenue predictions.
